# Algorithms & Data Structures

This repository contains implementations of common algorithms and data structures in TypeScript.

## Contents

- **Binary Search**: Efficient search algorithm for sorted arrays
- **Quicksort**: Fast sorting algorithm with divide and conquer approach
- **Linked List**: Dynamic data structure with nodes
- **Stack & Queue**: LIFO and FIFO data structures

## Usage

Each file contains the implementation with example usage. Run with Node.js or in a TypeScript environment.

```bash
npx ts-node binarySearch.ts
```

## Complexity Analysis

- Binary Search: O(log n) time, O(1) space
- Quicksort: O(n log n) average time, O(log n) space
- Linked List operations: O(1) for add/remove at ends, O(n) for search
- Stack/Queue: O(1) for push/pop operations
