# Portfolio

This is my personal portfolio website built with Next.js and Tailwind CSS.

## Features

- Responsive design
- Dark mode toggle
- Sections: Hero, About, Projects, Blog, Contact
- Downloadable CV

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Deploy

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
